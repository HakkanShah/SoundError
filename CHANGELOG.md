# Changelog

## 0.1.0

- Initial release.
- Added diagnostic watcher with debounced error-increase triggering.
- Added terminal and task failure classification engine.
- Added native sound playback with custom pack fallback.
- Added cooldown manager and mode handling.
- Added commands for testing, reloading sound packs, and toggling mode.
- Added unit tests, integration smoke tests, and CI workflow.
